# DR Tech Solutions – Membership Registration System

## 🌟 Overview

DR Tech Solutions Membership Portal is a simple, scalable, and user-friendly web application designed to streamline membership registration, career guidance support, and payment processing.

This platform enables individuals from various backgrounds to register, seek guidance, and enroll in programs with expert support.

## 🔄 Workflow

1. User fills the registration form
2. Data is stored in Google Sheets
3. Confirmation email is sent
4. User is redirected to Razorpay payment
5. Payment confirmation received via webhook
6. Admin can track all registrations

---

## 📊 Reports

* Total registrations tracked via Google Sheets
* Easy filtering and analytics
* Scalable for future dashboards

## 🚀 How to Use

1. Clone the repository
2. Update the following:

   * Google Apps Script URL
   * Razorpay Payment Link
3. Deploy via GitHub Pages
4. Connect Google Sheet backend
5. Start collecting registrations
